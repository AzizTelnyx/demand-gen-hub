## 🔍 Gaps

### 1. No Consistent BOFU Motion on LinkedIn
- Only a few BOFU campaigns (Twilio competitor, remarketing spotlight). Most budget is TOFU.
- Remarketing spotlight ads had great reach (470K impressions) but were paused — should be always-on.

### 2. Heavy Campaign Churn
- 84 campaigns in 12 months, majority now paused.
- High churn = short learning windows and wasted ramp-up spend.
- Many campaigns ran <$2K total — too small to learn or optimize.

### 3. No Vertical Consistency
- Healthcare, insurance, BPO, fintech, travel all tested but none scaled consistently.
- Pattern: launch vertical → run 4–6 weeks → pause → try new one.

### 4. Format Experimentation Without Structure
- SI, VA (video), CA (carousel), SPA (spotlight), MA (message ads) all used.
- No clear framework for which format fits which funnel stage.

### 5. Regional Coverage is Thin
- APAC has been expensive and underperforming. MENA barely present.
- Most budget concentrates on Global/AMER targeting.

### 6. No ABM Pipeline Attribution Connected
- LinkedIn is an ABM platform, but no domain-level pipeline influence data is being surfaced alongside these engagement metrics.
- The "conversions" column from LinkedIn's API counts platform events (opens, views, etc.) — not actual pipeline or revenue.

---

## Bottom Line

LinkedIn is working best as a **TOFU awareness engine for AI/Voice messaging to global audiences using single-image ads**. Anything niche-vertical, partnership-based, or video-format is bleeding money.

The biggest gaps:
1. **No structured always-on BOFU remarketing layer**
2. **No pipeline attribution to prove ROI**
3. **Campaign churn rate is too high** — need fewer, bigger bets that run long enough to optimize
