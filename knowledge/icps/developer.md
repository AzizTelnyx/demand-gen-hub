# ICP: Developer

## Overview
Software developers and engineering teams building communications features into applications.

## Firmographics
- **Company Size:** 10-10,000+ (wide range)
- **Role:** Backend developers, full-stack, DevOps, platform engineers
- **Industries:** SaaS, FinTech, HealthTech, Marketplaces, AI/ML startups
- **Geography:** Global, tech hubs (SF, NYC, Austin, London, Berlin, Tel Aviv)

## Personas

### Individual Developer
- Building side project or startup MVP
- Self-serve, credit card buyer
- Wants: Quick start, good docs, low cost
- Hates: Sales calls, complex pricing, vendor lock-in

### Team Lead / Engineering Manager
- Evaluating for company project
- Needs: Reliability, support, security compliance
- Decision maker for small-mid projects

### Platform/Infrastructure Engineer
- Building internal communications platform
- Enterprise needs, high volume
- Needs: SLAs, dedicated support, custom contracts

## Pain Points

### Developer Experience
- Poor documentation
- Slow API response times
- Complex SDKs and setup
- Limited language/framework support
- Unclear error messages

### Reliability
- Downtime affecting production
- Inconsistent message delivery
- Number provisioning delays
- Carrier issues

### Pricing
- Confusing per-message pricing
- Hidden fees (long code, short code, etc.)
- Unpredictable bills
- No volume discounts until enterprise

### Vendor Lock-in
- Proprietary APIs
- Number portability issues
- Difficult migration

## Buying Triggers

- New product requiring SMS/Voice
- Scaling issues with current provider
- Cost pressure (Twilio bill too high)
- Reliability incident with current provider
- Need for global coverage
- New compliance requirement

## Evaluation Criteria

### Must-Have
- Clear, complete API documentation
- Quick number provisioning
- Competitive pricing
- High deliverability rates
- Good support

### Nice-to-Have
- SDKs in preferred language
- Sandbox/test environment
- Webhook support
- Number pooling
- Analytics dashboard

## Competitors They're Evaluating

### Direct
- **Twilio** - Market leader, expensive
- **Vonage** - Enterprise focus
- **Bandwidth** - Network owner
- **Plivo** - Price competitor
- **MessageBird** - EU-focused

### Niche
- **Sinch** - Messaging focused
- **Infobip** - Enterprise messaging
- **SignalWire** - FreeSWITCH-based

## Typical Sales Cycle

### Self-Serve
- Length: Minutes to days
- Deal Size: $0-1K/month
- Process: Sign up, try, upgrade

### Team/Startup
- Length: 1-4 weeks
- Deal Size: $1K-10K/month
- Process: Trial, technical eval, procurement

### Enterprise
- Length: 1-3 months
- Deal Size: $10K-100K+/month
- Process: Security review, legal, procurement
